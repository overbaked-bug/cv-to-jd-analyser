# CV ↔ JD Gap Analyser

An AI-powered tool that scores your CV against a job description, 
surfaces gaps, and generates actionable interview talking points.

## What it does
- Match score (0–100) with a summary of your fit
- Gap analysis with severity levels (critical / moderate / minor) 
  and specific suggestions to address each gap in interviews
- Strong matches — maps your CV evidence to JD requirements
- Bullet rewrites — improves your weakest CV bullets for the role
- Talking points for recruiter calls

## How to use
1. Open the app
2. Enter your Anthropic API key (get one free at console.anthropic.com)
3. Paste or upload your CV (PDF supported)
4. Paste, upload, or fetch the JD from a URL
5. Click Analyse

## Tech
Built with vanilla HTML/CSS/JS and the Claude API (claude-sonnet-4). 
No frameworks, no build step — single HTML file.
